\# Arduino Source Code



This directory contains the Arduino program used to control the smart parking system.



\## File

\### `smart\_parking.ino`

Implements:

\- Vehicle detection using ultrasonic sensors

\- Circular Queue for vehicle entry management

\- Min Heap for nearest-slot allocation

\- Servo motor control for automated gate operation

\- Serial Monitor output for real-time status display



\## Logic Used

\- \*\*Circular Queue\*\*: Ensures FIFO vehicle entry handling

\- \*\*Min Heap\*\*: Allocates nearest available parking slot efficiently



\## Execution Steps

1\. Open `smart\_parking.ino` in Arduino IDE

2\. Select \*\*Arduino UNO\*\* board and correct COM port

3\. Upload the code to the board

4\. Open Serial Monitor to view system output
